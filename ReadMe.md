# Inventory & Order Management System (SQL Capstone)

## Overview
This project implements a relational database for an e-commerce inventory and order system.

## Tech Stack
- SQL (PostgreSQL)
- VS Code

## How to Run
1. Run `01_schema_ddl.sql`
2. Run `02_sample_data.sql`
3. Run `03_views.sql`
4. Run `05_stored_procedures.sql`
5. Run queries in `04_kpis_and_analytics.sql`

## Deliverables
- ERD Diagram
- DDL Scripts
- DML Queries
- Views & Stored Procedures
